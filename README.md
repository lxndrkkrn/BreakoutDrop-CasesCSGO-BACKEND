# Breakout-Drop : Cases CSGO
Проект BreakoutDrop v1.3

### Стек:
- Java 21, Spring Boot 4
- Spring Data JPA
- Lombok, Slf4j

### Функционал:
- Создание / Удаление пользователей
- Создание / Удаление кейсов
- Добавление / Удаление скинов
- Открытие кейсов со случайным (вашим) дропом
- Апгрейдинг скинов
- Контракт обмена
- Продажа скинов (отдельный / все сразу)

### Нововведения:
- Метод продажи скинов на баланс сайта

### Микроизменения:
- Добавлена возможность установки картинки для скина (через URL)

### В разработке:
- Вывод скинов в инвентарь Steam через ботов (в далёком светлом будущем...)
