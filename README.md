# Core Banking API
Проект BreakoutDrop v1.2

### Стек:
- Java 21, Spring Boot 4
- Spring Data JPA
- Lombok, Slf4j

### Функционал:
- Создание/Удаление пользователей
- Создание/Удаление кейсов
- Добавление/Удаление скинов
- Открытие кейсов со случайным (вашим) дропом
- Апгрейдинг скинов
- Контракт обмена

### Нововведения:
- Добавлена система "Контрактов обмена": Можно обменять от 3-х до 10-х скинов, на один другой (может быть дешевле или дороже на определённый коэфицент)

### Микроизменения:
- Исправлены мелкие баги при проверке баланса (при открытии кейсов или апгрейда оружия)
- Добавленны новые исключения
