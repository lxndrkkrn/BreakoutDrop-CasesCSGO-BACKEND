# Breakout-Drop : Cases CS2
Проект BreakoutDrop v1.4

### Стек:
- Java 21, Spring Boot 4
- Spring Data JPA
- Lombok, Slf4j

### Функционал:
- Создание / Удаление пользователей
- Создание / Удаление кейсов
- Добавление / Удаление скинов
- Открытие кейсов со случайным (вашим) дропом
- Апгрейдинг скинов
- Контракт обмена
- Продажа скинов (отдельный / все сразу)

### Нововведения:
- Метод пополнения баланса (для карт, а не продажа скина на баланс сайта)
- Сохранение всех транзакций
- "Общак" сайта
- Подтасовка скинов (Кейсы / Апгрейды / Контракты)

### Микроизменения:
- Исправлены мелкие баги
- Оптимизация

### В разработке:
- Вывод скинов в инвентарь Steam через ботов (в далёком светлом будущем...)
